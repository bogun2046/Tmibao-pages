TMiBao Support

TMiBao 2.4 is a local-first file and image redaction app. It supports Simplified Chinese, Traditional Chinese, English, and Japanese UI, following the system language. OCR and redaction run on the device, and confirmed results can be exported or shared.

Supported files and images

PDF DOCX TXT JPG PNG HEIC

These formats support redaction, export, and sharing.

Review and manual additions

OCR Manual review Box selection

OCR results should be reviewed manually, and missing areas can be added with manual box selection.

Interface language support

  • Version 2.4 supports Simplified Chinese, Traditional Chinese, English, and Japanese UI.
  • The app follows the system language by default.
  • Localization affects interface and help text only and does not change redaction logic.

Full workflow

Import ↓ OCR ↓ Review ↓ Export ↓ Share

After a file or image is imported, OCR runs locally on the device. The user reviews the result, can add extra redaction ranges manually, confirms the output, and then chooses whether to export or share it.

Currently supported sensitive content detection

Phone numbers ID numbers Names on ID cards Bank card numbers Email Amounts Court case numbers Labor arbitration case numbers Unified social credit codes Chinese addresses Japanese addresses

Automatic detection helps locate sensitive content. OCR results should still be reviewed manually, and users should supplement any missed content based on the actual file.

Local processing notes

  • Files and images are selected explicitly by the user.
  • All OCR and redaction processing happens on the device.
  • The app does not require sign-in and does not upload files, images, or PDFs.
  • The original file is not overwritten; the redacted result is created as a new copy.
  • Exporting and sharing are always user-initiated.

Known limitations

  • OCR accuracy depends on source file or image quality.
  • Handwriting recognition is limited.
  • Encrypted or damaged files may not be processable.
  • Complex layouts should be reviewed manually.
  • Blur, glare, skew, or low resolution may affect OCR results.
  • Redacted output should be checked again before external use.
  • Organization names are not default automatic redaction items; use manual selection or a custom rule when needed.
