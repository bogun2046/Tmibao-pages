TMiBao Privacy Policy

TMiBao is a local-first file and image redaction app. Content selected by the user is used only for on-device OCR, manual review, and redaction.

Local processing principles

  • All OCR and redaction processing happens on the device.
  • No user files are uploaded.
  • No images are uploaded.
  • No PDFs are uploaded.
  • No cloud OCR is used.
  • No cloud sync is used.
  • Redacted output is generated as a new copy and does not overwrite the original file.

How content enters the app

The app accesses content only when the user explicitly selects it. Files are imported with the system file picker. Images can be imported from the system photo picker, the file picker, or the camera. The app does not scan files or photos in the background.

Data not collected

  • User account information
  • Contact information
  • File and image content
  • Redacted file or image copies
  • Location, device identifiers, usage data, or diagnostics data

Services not used

  • Ad tracking
  • Cross-app or cross-site tracking
  • Third-party analytics SDKs
  • Third-party sign-in
  • Cloud OCR or cloud storage

Permissions and system capabilities

  • File picker: imports user-selected PDFs, DOCX, TXT files, or images.
  • Photo picker: selects images from the system photo library for image redaction.
  • Camera: requested only when the user chooses to take a photo for redaction.
  • Local file saving: saves the user-confirmed redacted result on the device.
  • Share sheet: opened only when the user chooses to share, and the user selects the destination.

The app does not request microphone, location, or contacts permissions.

OCR and image processing

The app supports local redaction for PDFs, DOCX, TXT, and images. OCR results may be affected by source quality, handwriting, skew, glare, or complex layouts, so manual review is recommended before export. For image redaction, the exported image writes redacted regions into pixels and removes EXIF, GPS, and device metadata from the original image.

Export and sharing

After redaction is complete, users can save the result locally or share it with the system share sheet. Exporting, saving, and sharing are always initiated by the user. The app does not send redacted results automatically.

Security and usage notice

The app reduces the need for files to leave the device by processing them locally. Even so, OCR and redaction results should still be checked by the user. The app does not guarantee detection of all sensitive information or 100% accuracy and does not replace legal, compliance, or other professional review.
