FAQ

Frequently Asked Questions

This page explains the public release behavior of TMiBao for file and image redaction, on-device processing, OCR, manual review, and exporting.

Which interface languages are supported?
Version 2.4 supports Simplified Chinese, Traditional Chinese, English, and Japanese UI. The app follows the device system language by default.
Can Chinese addresses be redacted?
Yes. The address rule recognizes Chinese addresses with labels such as “Address” or “Residence,” and Japanese addresses are supported as well. Review complex or unlabeled addresses manually and add a box selection when needed.
Are files uploaded?
No. User-selected files are processed only on the device and are not uploaded to developer servers or other cloud services.
Does it support image redaction?
Yes. JPG, PNG, and HEIC images are supported. OCR can help locate sensitive content, and the redaction area can be confirmed by manual review or manual box selection. The redacted result can then be exported or shared.
Can OCR-based results be exported formally?
Yes. PDFs and images can be exported after OCR recognition and manual review, and they can also be shared through the system share sheet.
Which file types are supported?
PDF, DOCX, TXT, plus JPG, PNG, and HEIC images are supported. PDFs include both text-layer PDFs and scanned or image-based PDFs that require OCR.
How accurate is OCR?
OCR accuracy depends on clarity, resolution, layout, skew, glare, and text type in the original file or image. The app does not guarantee 100% accuracy.
Why is manual review recommended?
OCR may miss content or misrecognize text. Complex layouts and handwriting can also affect results. Manual review helps confirm the redaction scope and lets users add missing areas with box selection.
Is sign-in required?
No. The app does not require account registration and does not include a sign-in flow.
Does it upload anything to a server?
No. All OCR and redaction processing happens on the device. No cloud OCR, cloud storage, or cloud sync is used.
Will the original file be overwritten?
No. The redacted result is generated as a new copy and the original file remains unchanged.
What permissions are needed for image redaction?
The photo picker is used when selecting from the system photo library. Camera permission is requested only when the user actively chooses to take a photo. Images are always processed on the device.
