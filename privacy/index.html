脱密宝隐私政策

脱密宝是一款本地文件脱敏工具。用户主动选择的文件会在设备本地进行识别和脱敏处理，用于生成脱敏副本，便于后续复核、保存或分享。

我们如何处理文件

  • 用户通过系统文件选择器主动选择文件。
  • 文件在设备本地完成处理。
  • App 生成脱敏副本，不覆盖原文件。
  • 用户可自行决定是否导出或分享脱敏副本。

我们是否上传文件

当前版本不会将用户选择的文件上传到开发者服务器。当前版本不提供云端同步、云端 OCR、云端存储或服务器侧文件处理能力。

当前不收集的数据

  • 用户账户信息
  • 联系方式
  • 文件内容
  • 脱敏后的文件副本
  • 位置、标识符、使用数据、诊断数据

当前不使用的能力

  • 广告跟踪
  • 跨 App / 跨网站追踪
  • 第三方分析 SDK
  • 第三方登录
  • 云端 OCR 服务

权限与系统能力说明

当前版本可能使用文件选择器 / 文档访问、分享面板和本地文件保存。当前版本不主动请求相机、相册、麦克风、定位、通讯录等权限。

OCR PDF 能力边界

当前版本正式支持 TXT、DOCX 和文字层 PDF 的脱敏处理。OCR PDF / 扫描型 PDF 当前仅作为草稿 / 实验能力展示，不开放正式脱敏导出。

数据安全与使用限制

本应用尽量将文件处理限制在设备本地，以减少文件内容离开设备的场景。但请注意，脱敏结果仍需用户自行复核。本应用不承诺识别所有敏感信息，不承诺识别结果 100% 准确，也不替代法律、合规或其他专业审查。
